# Pre-Launch Checklist

## Code Quality
- [ ] All console.logs removed from production code
- [ ] No commented-out code
- [ ] Code is properly formatted
- [ ] No unused imports
- [ ] All TODOs are resolved

## Assets
- [ ] App icon is finalized (1024x1024)
- [ ] Splash screen is finalized
- [ ] All images are optimized
- [ ] App name is correct
- [ ] Bundle identifier is correct

## Configuration
- [ ] app.json is complete
- [ ] Environment variables are set
- [ ] API endpoints are correct
- [ ] Push notification setup is complete
- [ ] App version number is correct

## Testing
- [ ] All tests pass
- [ ] Tested on multiple devices
- [ ] Tested on iOS and Android
- [ ] Performance is acceptable
- [ ] No critical bugs

## Legal
- [ ] Privacy policy is written
- [ ] Terms of service are written
- [ ] App store descriptions are ready
- [ ] Screenshots for app stores are ready
- [ ] Contact information is provided

## Backend
- [ ] Database is optimized
- [ ] All RLS policies are tested
- [ ] Backups are configured
- [ ] Monitoring is set up
- [ ] Rate limiting is configured

## Documentation
- [ ] README is complete
- [ ] Installation guide is clear
- [ ] API documentation exists
- [ ] User guide is written
