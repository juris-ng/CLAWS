# App Store Submission Checklist

## Apple App Store

### Required Assets
- [ ] App Icon (1024×1024px, PNG, no transparency)
- [ ] iPhone Screenshots (6.7" display - 1290×2796px)
- [ ] iPhone Screenshots (5.5" display - 1242×2208px) 
- [ ] iPad Screenshots (12.9" - 2048×2732px) - if supporting iPad
- [ ] App Preview Video (optional, 15-30 seconds)

### App Information
- [ ] App Name (30 characters max)
- [ ] Subtitle (30 characters max)
- [ ] Description (4000 characters max)
- [ ] Keywords (100 characters, comma separated)
- [ ] Support URL
- [ ] Marketing URL (optional)
- [ ] Privacy Policy URL
- [ ] Category (Primary and Secondary)
- [ ] Age Rating

### Build Information
- [ ] Bundle ID configured
- [ ] Version number set
- [ ] Build number set
- [ ] Signing configured
- [ ] Push notification certificates

### Review Information
- [ ] Demo account credentials (if needed)
- [ ] Contact information
- [ ] Notes for reviewer

## Google Play Store

### Required Assets
- [ ] App Icon (512×512px, PNG)
- [ ] Feature Graphic (1024×500px)
- [ ] Phone Screenshots (min 2, max 8)
- [ ] 7-inch Tablet Screenshots (optional)
- [ ] 10-inch Tablet Screenshots (optional)
- [ ] Promo Video (YouTube URL, optional)

### Store Listing
- [ ] App Name (50 characters max)
- [ ] Short Description (80 characters max)
- [ ] Full Description (4000 characters max)
- [ ] Category
- [ ] Tags/Keywords
- [ ] Contact Email
- [ ] Privacy Policy URL
- [ ] Website URL (optional)

### Content Rating
- [ ] Complete content rating questionnaire
- [ ] Target age group
- [ ] Content descriptors

### App Release
- [ ] Signed APK/AAB
- [ ] Version name
- [ ] Version code
- [ ] Release notes
- [ ] Rollout percentage (start with 10-20%)

## Both Stores

### Legal
- [ ] Privacy Policy published and linked
- [ ] Terms of Service published
- [ ] Copyright information
- [ ] Data safety information
- [ ] Export compliance

### Testing
- [ ] App tested on multiple devices
- [ ] All features working
- [ ] No crashes or major bugs
- [ ] Performance is acceptable
- [ ] Screenshots match actual app

### Localization (if applicable)
- [ ] Translated descriptions
- [ ] Localized screenshots
- [ ] Translated keywords
