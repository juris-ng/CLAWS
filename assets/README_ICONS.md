## Design Guidelines

### Color Scheme
- Primary: #0066FF (Blue)
- Secondary: #34C759 (Green)
- Accent: #FFD700 (Gold)
- Background: #FFFFFF (White)

### Icon Style
- Minimalist and modern
- Clear and recognizable at small sizes
- Represents civic engagement/democracy
- Avoid text in icon (use symbols instead)
